# **App Name**: PES Nexus

## Core Features:

- Syllabus Grounding: The AI is anchored to the PESMCOE syllabus, ensuring that explanations are strictly within the college's defined scope. This is implemented by using the syllabus documents as a tool that the LLM can consult.
- Curated Resource Mapping: The AI provides curated YouTube keywords and reference book suggestions for each unit of the syllabus.
- Senior-Nexus Bridge: A system where students can flag difficult autonomous topics for senior mentorship.
- Autonomous Topic Input: Allow students to input specific autonomous topics for explanations.
- Resource Display: Display curated YouTube keywords and reference book suggestions in a clear, organized manner.

## Style Guidelines:

- Primary color: Dark Gray (#A9A9A9) to reflect the intellectual and academic nature of the application, reminiscent of traditional educational materials.
- Background color: Black (#000000), nearly desaturated blue, for a clean, distraction-free interface, ensuring readability and focus.
- Accent color: White (#FFFFFF), an analogous color that conveys a sense of trust and support, complementing the primary blue and enhancing key interactive elements without overwhelming the user.
- Headline font: 'Space Grotesk', a modern sans-serif, is used to maintain a computerized, techy feel; Body font: 'Inter', a grotesque-style sans-serif is used to increase readability in larger bodies of text
- Code font: 'Source Code Pro' for displaying code snippets clearly.
- Simple, geometric icons to represent different topics and resources.
- Clean and organized layout with clear sections for topic explanations, resources, and mentorship requests.
- Subtle animations for loading and displaying information to enhance user experience.
